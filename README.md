# InsertOneWord Companion

英単語ショート動画「InsertOneWord」の復習用Webアプリです。

動画で見た単語を、短時間で繰り返し確認できます。

---

## 🔤 機能

- 英単語＋日本語表示
- 「覚えた / まだ」で管理
- 「まだ」を優先したランダム出題
- 単語ごとの動画リンク
- PWA対応（ホーム画面追加可）

---

## 📱 使い方

1. 単語を確認
2. 「覚えた」または「まだ」を選択
3. 次へ進む

スキマ時間の復習を想定しています。

---

## 🛠 技術スタック

- Vanilla JavaScript (ES Modules)
- localStorage
- PWA
- GitHub Pages

---

## 🎯 目的

英単語学習の継続性を高めるためのミニマルな復習体験を提供します。
